## **Setup ⚙️**<a name="setup"></a>

Clone this repository:<br>
```sh
  git clone https://github.com/mariegrace31/portfolio_project.git

```
Navigate to the folder
```
cd portfolio_project
```

```
git checkout main
```

npx eslint . --fix
